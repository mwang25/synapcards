Cards

  1. Top row of all cards should have card id on the left, and icons in the middle, and a drop down menu on the right. Card is is a hyperlink to the card itself.
  2. Heart icon for likes. Mouse over like icon to see how many likes. Should author of the card be allowed to like his own card? And should the icon look different if the author has liked it?
  3. Eye icon for number of views. Too fine grained?
  4. If this card is not owned by the viewer, then a bell icon (or telescope) for getting notified of updates to this card. Too fine grained? Is following the user good enough?

Cards can be private, public, or shared, meaning view permissions granted to specific users. Should we ever allow other people to write to someone else's card? Seems hard to deal with. They could just create a shared account.

Card Drop down menu

  1. If this card is owned by the viewer, then sharing option. Sharing allows the owner to make the card private, share with other synapcard users, or make the card public. Instead of putting this in the drop down menu, configure it within the context of the current add/delete/edit buttons and form?
  2. If this card is not owned by the viwer, then copy option. Copy will create a new card owned by the current viewer. Once the copy is made, the copy is independent of the original and can diverge from the original card.
  3. Everyone will have a meta info option, which shows meta info about the card, such as creation date, last update date, etc.

Login

Support for users logging into their own accounts using oath/google identification. Login information is needed to implement secure email updates, sharing, and support for multiple users. (done 11/25/17)

Each user can optionally create a profile page for him/herself. Profile page can also show stats about the account, e.g. how many cards the user has created, how many added last month, distribution of tags, total views(?), total likes(?), when the user joined synapcards, etc. Need at least one setting: timezone the user lives in. (Most of the information is there on the user's profile/settings page. Add more stuff, e.g. tags distribution as time goes on. 11/25/17)

Privacy

Every card will have privacy control. They can be set to private, shared with specific synapcard users, or public.
